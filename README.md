# VisitTrack Pro

## 功能说明
*   **客户拜访记录**: 支持 10 张高清照片上传，完整记录拜访细节。
*   **地理定位**: 一键获取 GPS 坐标，并生成 Google Maps 实时导航链接。
*   **云端同步**: 数据实时保存至 Google Sheets，支持多终端同步查看。
*   **默认日期**: 记录日期默认为“今天”，提高录入效率。

## 🚀 部署指南

### 后端 Google Apps Script 结构 (V8)
请在部署脚本时确保您的 Google Sheet 表头与下方定义一致：

| 索引 | 字段名 | 说明 |
| :--- | :--- | :--- |
| 0 | 地区 | 客户所属大区或具体地址 |
| 1 | Google定位 | 地图链接 |
| 2 | 客户名 | 核心标识字段 |
| 3 | 拜访日期 | YYYY-MM-DD |
| 4 | 拜访记录 | 详细文字记录 |
| 5-14 | 照片1-10 | ImgBB 存储链接 |
| 15 | ID | 系统唯一标识 (时间戳) |
| 16 | Lat | 纬度 |
| 17 | Lng | 经度 |
| 18 | AI_Analysis | AI 分析预留位 |

## 系统配置
1.  **ImgBB**: 请在设置中填入 API Key。
2.  **GAS URL**: 请填入部署为 Web 应用后的 `/exec` 链接。
